# Tienda Online

Gestor de tienda virtual.

## Base de datos

Restaurar la base de datos desde el archivo script.sql desde **MySQL Workbench**, o phpMyAdmin, o desde el intérprete de comandos.


## API

La carpeta `tienda_api` contiene un proyecto Node.JS con la API que permite recuperar y almacenar datos en una base de datos tipo MySQL.

Instalar las dependencias con el comando desde la carpeta:

    npm install

Una vez instaladas las dependencias se debe ejecutar con el comando:

	npm start

## Front-end

La carpeta `tienda_frontend` contiene el proyecto en Angular para facilitar la interacción del usuario en la tienda.

Para la instalación de las dependencias se utiliza el comando:

    npm install

Una vez instaladas las dependencias se debe ejecutar con el comando:

	npm start
